## 核心目录说明

### `/src` 源代码目录
- **components/**: 可复用的 Vue 组件
- **layouts/**: 页面布局组件
- **pages/**: 页面级组件
- **plugins/**: Vue 插件配置
- **router/**: Vue Router 路由配置
- **stores/**: Pinia 状态管理
- **styles/**: 全局样式文件
- **assets/**: 静态资源文件

### `/rules` 项目规则目录
- 包含项目开发规范和约定
- 代码编写规则和最佳实践
- 项目文件组织结构说明

### `/test` 测试目录
- 单元测试文件
- 组件测试文件
- 集成测试文件

## 技术栈

- **框架**: Vue 3 + TypeScript
- **UI 库**: Vuetify 3.x
- **构建工具**: Vite
- **路由**: Vue Router
- **状态管理**: Pinia
- **测试**: Vitest + Vue Test Utils
- **代码规范**: ESLint + Prettier

## 开发规范

1. **文件命名**: 使用 PascalCase 命名 Vue 组件文件
2. **目录结构**: 按功能模块组织代码
3. **组件设计**: 基于 Vuetify 3.x 组件库开发
4. **类型安全**: 全面使用 TypeScript 类型定义
5. **测试覆盖**: 使用 Vitest + Vue Test Utils 进行组件测试
